(static) either(val1, val2, function) → {*}

Source:
  • ext.js, line 31
Determines the correct value to return, by calling the passed in check function.
If no check function, then it uses the softFalsy method.
Examples
either(0, 2)
// Returns 0
either(null, 2)
// Returns 2
either(1, 2, (val1, val2) => { return true })
// Returns 1
Parameters:
Name Type Description
val1 * return if passes in check method return true
val2 * return if passed in check method returns false
